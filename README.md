#DESCRIPTION:This project implements a real-time multithreaded chat application using Java Sockets. The system follows a clientâ€“server architecture, where a central server handles multiple clients at the same time using multithreading.
The server continuously listens for client connections. When a client joins, the server creates a separate thread to handle that client's messages. This allows many users to chat simultaneously without interruption.
The client program connects to the server using a socket. Users can type messages, which are sent to the server. The server then broadcasts these messages to all connected clients, enabling group chat functionality.
